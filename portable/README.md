# Portable Blindspot

Blindspot for any LLM — no Claude Code, no plugin, no filesystem. Just a prompt and a log you keep yourself.

- **[blindspot-prompt.md](blindspot-prompt.md)** — the full mirror (~8k chars). Paste it at the end of any work conversation, or install it as project/Gem/Custom-GPT instructions.
- **[blindspot-compact.md](blindspot-compact.md)** — condensed card (~1.6k chars) for small custom-instruction fields.

Quick start: finish a work session in any chat, paste `blindspot-prompt.md` into the same conversation, read what comes back, and save the final fenced block into a private note. Next time, paste that note in first.

Full per-platform instructions (ChatGPT, Claude.ai, Gemini, Copilot, local models) and the log ritual: [docs/any-llm.md](../docs/any-llm.md).

These files must stay behaviorally in sync with the Claude Code skill at [skills/blindspot/SKILL.md](../skills/blindspot/SKILL.md) — same evidence rules, same strain tier, same distress/self-harm hard stop. If you change one, change the others.
